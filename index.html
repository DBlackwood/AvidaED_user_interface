Digital Evolution for Education

Avida-ED is an award-winning educational application developed at Michigan State University for undergraduate biology courses to help students learn about evolution and scientific method by allowing them to design and perform experiments to test hypotheses about evolutionary mechanisms using evolving digital organisms.

Avida-ED Application

Posted on 2017-02-22 by Diane Blackwood

Avida-ED 3.2.1

Web Hosted

  • Avida-ED 3.2.1 web application

Hints and Suggestions

  • Change font size: using on a Mac <command> <+> and <command> <->
  • Change font size: using on a PC <control> <+> and <control> <->

Best Run In

  • Firefox 46.0.1 and later
  • Google Chrome 51.0.2704.84 and later

Downloadable Versions

For working where there is no internet access

  • Avida-ED Windows installer for self-hosted version 2016_0918
  • Avida-ED 3.0.0 for Mac OS 10.6 and above
    • Unzip the file once it is downloaded
    • Move the file ‘Avida-ED Web.app’ to the Applications Folder
    • . . . Your Mac will complain that the file is from an unknown source, you will have to approve opening it

Known Problems

  • Cannot save workspace
    • Safari
  • Dragging an organism onto the Grid – the user must click on the grid destination before dragging the organism to the grid
    • Google Chrome 56.0 and later
    • IE
    • Touch Screens
