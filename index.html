    • Mac OS X self-hosted version

    The DMG linked above includes all the needed files to run Avida-ED Web locally on a Mac OS X machine. To use this, follow these steps:

    1. Download the DMG.
    2. Open the DMG on the host machine.
    3. Copy all the contents of the DMG to a local directory.
    4. Open a terminal window and change directory to be in the local directory with the copied files.
    5. Issue this command in the terminal: 
      ./avServer
    6. If Safari is your default browser, copy the URL of the window that opened, open Firefox or Chrome, and paste in the URL.
